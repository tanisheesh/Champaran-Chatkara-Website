# Champaran Chatkara

A modern restaurant website for authentic Bihari cuisine, bringing the traditional flavors of Champaran to the digital world.

## About

Champaran Chatkara is a family-owned restaurant serving authentic Bihari dishes for over 25 years. Located in Kaushambi, Ghaziabad, we specialize in traditional recipes passed down through generations, featuring signature dishes like Litti Chokha and Champaran Mutton.

## Features

- **Responsive Design** - Optimized for all devices from mobile to desktop
- **Interactive Menu** - Browse our complete menu with search and category filters
- **Smooth Animations** - Engaging user experience with fluid transitions
- **Admin Dashboard** - Content management system for menu and restaurant data
- **Modern UI** - Clean, professional design with authentic Bihari aesthetics

## Tech Stack

- **Framework**: Next.js 16 with TypeScript
- **Styling**: Tailwind CSS with custom animations
- **Database**: Supabase for data management
- **UI Components**: Radix UI primitives
- **Authentication**: Supabase Auth for admin access
- **Deployment**: Optimized for production builds

## Pages

- **Home** - Hero section, about us, special dishes, and contact information
- **Menu** - Complete restaurant menu with search and filtering capabilities
- **Admin** - Protected dashboard for managing menu items and categories
- **Admin Login** - Secure authentication for restaurant staff
